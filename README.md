Система агрегації та структурування державних актів

Даний проєкт присвячений створенню інтелектуальної платформи для роботи з державними нормативно-правовими актами України. 

**Роль та бачення**
У цьому презентуючому портфольному проєкті я поєдную компетенції **Business Analyst**, **Fullstack Developer** та **QA Engineer**. Моя мета — трансформувати масиви неструктурованого юридичного тексту в динамічну об'єктну модель, що забезпечує:
* *Цілісність* даних: Автоматичне відстеження ієрархічних зв'язків та змін у редакціях.
* *Прозорість*: Візуалізація структури нормативних актів для швидкого аналізу.
* *Доступність*: Створення інтуїтивного інтерфейсу для ефективної взаємодії з державними даними.

**Технологічний стек**
* *Frontend:* HTML5 / CSS3 / JavaScript (з перспективою переходу на React.js).
* *Backend:* C# (.NET 8) — обрано для побудови надійної архітектури корпоративного рівня.
* *Database:* MySQL — для зберігання складних реляційних зв'язків між актами.
* *QA & Testing:* Unit Testing (xUnit/NUnit), Integration Testing, E2E testing логіки парсингу.
* *Concepts:* проектування архітектурних шарів (Business, Application, Technology) та візуалізація взаємодії компонентів системи.
