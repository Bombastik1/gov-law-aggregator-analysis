# Матриця ризиків проєкту GovLaw

Цей документ містить аналіз потенційних загроз та план реагування на них. Повна версія розрахунків доступна у прикріпленому файлі [Excel](./Мatrix_of_risks.xlsx).

| Ризик | Ймовірність | Вплив | Стратегія реагування |
| :--- | :---: | :---: | :--- |
| **Технічний:** Нестабільність API (data.gov.ua) | Висока | Критичний | Впровадження механізму кешування та черг запитів. |
| **Аналітичний:** Складна структура PDF-актів | Середня | Високий | Використання бібліотек розпізнавання тексту (OCR) та ручна перевірка. |
| **Юридичний:** Зміна законодавства про доступ до даних | Низька | Критичний | Моніторинг нормативної бази та адаптація модулів збору. |
| **Операційний:** Перевищення лімітів запитів до БД | Середня | Середній | Оптимізація запитів MySQL та індексація таблиць. |
