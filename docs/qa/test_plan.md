# Test Plan: GovLaw Aggregator

1. Функціональне тестування (Manual)
* *TC-01:* Перевірка пошуку закону за ключовим словом (наприклад, "Земельний").
* *TC-02:* Перевірка відображення списку актів при завантаженні головної сторінки.
* *TC-03:* Перевірка переходу на сторінку конкретного закону.

2. API Тестування (Backend)
* *TC-04:* Перевірка методу `GET /api/laws` — статус 200 OK.
* *TC-05:* Перевірка формату JSON-відповіді від сервера.
* **TC-06:** Перевірка обробки помилки при запиті неіснуючого ID (404 Not Found).

3. UI/UX Тестування
* *TC-07:* Перевірка адаптивності інтерфейсу (відображення на мобільних пристроях).
* *TC-08:* Перевірка швидкості рендерингу карток законів.
